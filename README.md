# GUI Calculator app using TKINTER

## This is a simple calculator app created using the GUI library **Tkinter** with python 

It supports many functions namely :-

* Addition 

* Substraction

* Multiplication 

* Division 

* Exponents

* Percentage

* Special Multiplication with **PI**

* Factorial
